# 🌍 EcoStride — Gamified Carbon Footprint Tracker

## 🔗 Repository
Public GitHub link: [Carbon Footprint Repo](https://github.com/Guptha747458/carbon-footprint.git)

---

## 📌 Chosen Vertical
**Sustainability & Climate Action**  
EcoStride is designed to help individuals understand, track, and actively reduce their personal carbon footprint in an engaging, gamified way.

---

## ⚙️ Approach & Logic
- **Prompt-driven development**: Built using Google AI tools with vibe coding (prompt-to-production flow).
- **Gamification**: XP, levels, streaks, and eco-habit logging to make climate action fun and rewarding.
- **Data-driven insights**: Benchmarks against global/EU/US averages to contextualize personal impact.
- **Simulation**: “What-If” scenarios to visualize lifestyle changes and their effect on emissions.

---

## 🛠️ How the Solution Works
1. **Interactive Carbon Calculator** — Users input transport, energy, diet, and waste data.
2. **Personalized Dashboard** — Visual charts (doughnut + bar) compare personal footprint vs benchmarks.
3. **Action Hub** — Log eco-friendly habits, earn XP, track streaks, and see monthly savings.
4. **What-If Simulator** — Experiment with lifestyle changes to see potential impact.
5. **Authentication + Persistence** — MongoDB backend ensures progress is saved securely.
6. **UI/UX** — Clean, modern interface with dark/light mode toggle.

---

## 🧩 Assumptions Made
- Average emission factors are based on publicly available datasets (global/EU/US).
- User inputs are self-reported and approximate; accuracy depends on honesty and detail.
- Gamification metrics (XP, streaks, levels) are designed for motivation, not scientific precision.
- The solution assumes internet access for deployment (Vercel hosting) and database connectivity.

---

## 🚀 Tech Stack
- **Frontend**: React + Vite  
- **Backend**: Express/Node.js + MongoDB  
- **Deployment**: Vercel  
- **AI Workflow**: Google AI vibe coding (prompt-to-production)
